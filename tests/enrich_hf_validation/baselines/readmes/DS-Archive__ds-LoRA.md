# Doctor Shotgun's Networks

This repository will serve as a location for uploading my personal LoRAs, LyCORIS, and Dreambooth models for Stable Diffusion 1.5.

My CivitAI: https://civitai.com/user/DocShotgun/models

Uploads to this repository will be without extensive preview image embeds to keep the main page clean.

# Model Description and Usage

Models are separated between standard LoRAs and LyCORIS (LoCon, LoHa, IA3, LoKr) networks.

Automatic1111's WebUI now has native LoRA support. Install them in .\stable-diffusion-webui\models\Lora\ and activate them by prompting \<lora:name_of_your_lora:network_weight\>

For LyCORIS models, Kohaku's extension is required for use (As of A1111 version 1.5.0+, LyCORIS support is integrated): https://github.com/KohakuBlueleaf/a1111-sd-webui-lycoris

LyCORIS files are placed in .\stable-diffusion-webui\models\LyCORIS\ and activated with \<lyco:name_of_your_lycoris:network_weight\>

Style networks are trained without trigger words required.

# Training Details

LoRAs trained locally using kohya_ss sd-scripts:

https://github.com/kohya-ss/sd-scripts

Several GUI options exist for sd-scripts including Derrian Distro's easy training scripts and bmaltais' GUI:

https://github.com/derrian-distro/LoRA_Easy_Training_Scripts

https://github.com/bmaltais/kohya_ss

Kohaku's extension is required for training LyCORIS models:

https://github.com/KohakuBlueleaf/LyCORIS
